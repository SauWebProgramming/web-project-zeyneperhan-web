# SAÜ Web Programlama Vize Projesi: Film Veri Görüntüleme Uygulaması
## Proje Açıklaması
Bu proje, web programlama dersi vize ödevi kapsamında geliştirilmiş, statik JSON verilerini kullanarak dinamik bir film kataloğu görüntüleme uygulamasıdır. Kullanıcıların filmlere ait temel bilgilere kolayca ulaşmasını sağlar.
## Kullanılan Teknolojiler
* **HTML5** (dizin.html): Uygulamanın temel iskeleti.
* **CSS3** (stil.css): Uygulamanın görsel tasarımı ve stillendirmesi.
* **JavaScript** (javascript.js): Filmlerin dinamik olarak yüklenmesi ve sayfaya işlenmesi.
* **JSON** (movies.json / filmlerjson): Film verilerinin saklandığı format.
